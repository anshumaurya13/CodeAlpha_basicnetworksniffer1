# CodeAlpha Basic Network Sniffer

## Objective

To capture and analyze network packets using Python and Scapy.

## Features

* Captures live network traffic
* Displays Source IP and Destination IP
* Identifies TCP, UDP and ICMP protocols
* Shows packet payload information

## Technologies Used

* Python
* Scapy

## How to Run

Install Scapy:

pip install scapy

Run:

python network_sniffer.py

## Learning Outcomes

* Understanding packet structures
* Network protocol analysis
* Packet sniffing fundamentals
